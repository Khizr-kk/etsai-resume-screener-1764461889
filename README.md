# FresherPath AI

Auto-generated by ETS-AI 10x.

## Idea Summary
An AI-powered resume screener for Indian fresher software engineers that analyzes resumes, compares them with job descriptions, and provides a score along with missing skills.

## Problem Statement
Indian freshers applying for software jobs struggle to optimize their resumes for specific job descriptions and identify skill gaps, hindering their job search.

## Target Persona
Indian fresher software engineer

## MVP Features
- User authentication and profile management
- Resume upload (PDF, DOCX)
- Job description upload/paste
- AI-powered resume-JD compatibility scoring (0-100)
- Identification and listing of missing skills based on JD
- Basic dashboard to view analysis results
- Free tier for one basic resume score and 1-2 top missing skills
- Premium tier for detailed analysis and full skill gap report

## Tech Stack
Python (FastAPI/Django) for backend and ML services, React/Vue.js for frontend, PostgreSQL for database, Docker for containerization, AWS/GCP for cloud infrastructure. ML libraries like Scikit-learn, spaCy, and potentially Hugging Face Transformers for advanced NLP.
